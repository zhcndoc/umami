---
title: 标签
---

使用 Umami 标签将事件分组到同一个网站下，收集数据，并启用各种功能。

## 用例

- A/B 测试：测试网页或活动的不同版本，查看哪个表现更好。
- 事件分组：允许在单个网站概览下进行过滤和洞察。

## 使用方法

在跟踪脚本中添加 **data-tag** 属性。这将在数据负载中包含 **tag** 列。

```html
<script
  defer
  src="http://mywebsite.com/umami.js"
  data-website-id="94db1cb1-74f4-4a40-ad6c-962362670409"
  data-tag="homepage-layout-a"
></script>
```

你可以在网站的不同部分重复使用跟踪脚本，添加不同的标签。

```html
<script
  defer
  src="http://mywebsite.com/umami.js"
  data-website-id="94db1cb1-74f4-4a40-ad6c-962362670409"
  data-tag="homepage-layout-b"
></script>
```

## 过滤

在网站 **概览** 页面，你可以按特定标签进行过滤。

![image](/images/docs/tag-filters.png)

点击任何指标卡上的 **更多**，也可以查看按标签划分的页面浏览细分情况。
