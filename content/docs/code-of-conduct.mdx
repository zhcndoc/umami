## 我们的承诺

作为成员、贡献者和领导者，我们承诺为社区的每个人提供一个免受骚扰的参与环境，
无论其年龄、体型、显性或隐性残疾、种族、性特征、性别认同及表达、经验水平、教育背景、社会经济地位、
国籍、个人外貌、种族、种姓、肤色、宗教或性别身份和性取向。

我们承诺以有助于营造开放、包容、多元、欢迎且健康社区的方式行动和互动。

## 我们的标准

有助于营造社区积极环境的行为示例包括：

- 对他人表现出同理心和善意
- 尊重不同的观点、看法和经历
- 乐于给予并优雅地接受建设性反馈
- 承担责任，向因我们过失而受到影响的人道歉，并从中学习
- 关注不仅对个人有利，更对整个社区有益的事情

不可接受的行为示例包括：

- 使用性暗示的言语或图像，或任何形式的性骚扰或性骚扰行为
- 钓鱼、侮辱或贬低性言论，以及针对个人或政治的攻击
- 公开或私下的骚扰
- 未经明确许可公开他人的私人信息，如住址或电子邮箱
- 其他在职业环境中合理被视为不当的行为

## 执行职责

社区领导负责明确和执行我们的可接受行为标准，并将对任何被认为不当、威胁性、冒犯性或有害的行为采取适当且公正的纠正措施。

社区领导有权且有责任删除、编辑或拒绝与本行为准则不符的评论、提交、代码、维基编辑、问题及其他贡献，并在适当情况下说明审核决策的理由。

### 适用范围

本行为准则适用于所有社区空间，也适用于个人在公开场合正式代表社区时。代表社区的示例包括使用官方电子邮箱地址、通过官方社交媒体账号发布内容，或作为指定代表参加线上或线下活动。

### 执行

对辱骂、骚扰或其他不可接受行为的举报，请联系负责执行的社区领导，邮箱：[contact@umami.is](mailto:contact@umami.is)。所有投诉都会被及时且公正地审查和调查。

所有社区领导必须尊重举报人的隐私和安全。

### 执行指南

社区领导在判定任何违反本行为准则的行为后，将遵循以下社区影响指南来决定相应的后果：

#### 1. 纠正

社区影响：使用不当用语或其他被认为不专业或社区不欢迎的行为。

后果：社区领导发出私人书面警告，明确说明违规性质并解释行为为何不当。可能要求公开道歉。

#### 2. 警告

社区影响：通过单一事件或一系列行为构成违规。

后果：警告及继续违规的后果。在指定时间内不得与相关人员互动，包括不得未经请求与执行行为准则的人员互动。此限制适用于社区空间及外部渠道，如社交媒体。违反条款可能导致暂时或永久禁止参与。

#### 3. 临时禁令

社区影响：严重违反社区标准，包括持续的不当行为。

后果：在指定期限内，禁止与社区进行任何形式的互动或公开沟通。期间不得与相关人员进行公开或私下的互动，包括未经请求与执行者互动。违反此条款可能导致永久禁令。

#### 4. 永久禁令

社区影响：表现出持续违反社区标准的行为模式，包括持续的不当行为、骚扰个人或对特定群体进行攻击或贬损。

后果：永久禁止在社区内进行任何形式的公开互动。
