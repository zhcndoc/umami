---
title: Boards
---

*Available since v3.1.0*

Boards are customizable dashboards that let you arrange analytics widgets in a flexible grid layout.
You can create boards scoped to a specific website, pixel, or link, or mix components from multiple sources in a single view.
Boards can be shared with team members or made public via a share link.

## Board types

| Type | Description |
| --- | --- |
| `mixed` | Open board with no required entity. Mix components from multiple websites, pixels, and links. |
| `website` | Scoped to a specific website. Unlocks website-only components. |
| `pixel` | Scoped to a specific pixel. Displays metrics collected by that pixel. |
| `link` | Scoped to a specific link. Displays click and traffic metrics for that link. |

## Add a board

Log into Umami and click on **Boards** in the sidebar.

{/* ![image](/images/docs/boards-sidebar.png) */}

Click on the **Add board** button in the top-right corner and select a board type.

![image](/images/docs/board-add.png)

Fill out the form with a name and optional description, then click **Save**.

{/* ![image](/images/docs/boards-add-form.png) */}

## Add components

Open a board and click **Edit** to enter edit mode. Click the **Add row** button to insert a new row, then click inside any empty column to select a component.

{/* ![image](/images/docs/boards-edit.png) */}

Each row can hold up to **4 columns**. Row heights and column widths are resizable by dragging the dividers.

{/* ![image](/images/docs/boards-resize.png) */}

## Components

| Component | Category | Description | Board types |
| --- | --- | --- | --- |
| Website Metrics Bar | Overview | Summary bar showing views, visitors, bounces, and time on site. | All |
| Website Chart | Overview | Line chart of page views and visitors over time. | All |
| Metrics Table | Tables | Configurable table of metrics by dimension type (pages, referrers, browsers, countries, etc.) with adjustable row limits. | All |
| World Map | Visualization | Geographic distribution of visitors plotted on a world map. | All |
| Weekly Traffic | Visualization | Heatmap of traffic intensity by day of week and hour. | All |
| Events Chart | Visualization | Line chart of custom events over time. | Website only |
| Text Block | Content | Free-form text for labels, annotations, or descriptions. | All |

## Share a board

From the board view click the **Share** button to generate a public share link. Anyone with the link can view the board without logging in.

{/* ![image](/images/docs/boards-share.png) */}

## Edit or delete a board

From the **Boards** screen click the **Edit** button to update a board's name, description, or layout, or click the **Delete** button to remove it.

{/* ![image](/images/docs/boards-table.png) */}
