---
title: 在 Netlify 上运行
---

[Netlify](https://netlify.com/) 提供了一项免费的托管服务，非常适合 GitHub 组织。

在此配置中，您应已经有一个数据库服务器在运行并接受远程连接。如果您还没有数据库，可以按照 [在 DigitalOcean 上运行](/docs/running-on-digitalocean) 指南来启动数据库。您也可以查阅 [托管](/docs/guides/hosting) 下的 **托管数据库** 部分。

## 设置

[![使用 netlify 部署](https://www.netlify.com/img/deploy/button.svg)](https://app.netlify.com/start/deploy?repository=https://github.com/umami-software/umami)

_使用上方按钮自动执行步骤 1-5_

1. 将项目 [https://github.com/umami-software/umami](https://github.com/umami-software/umami) fork 到您的 GitHub 账号。
2. 在 [Netlify](https://netlify.com/) 注册一个账户。
3. 在仪表盘页面点击 **导入项目**，然后指定您在 GitHub 上 fork 的项目的 URL。
4. 将构建命令更改为 **yarn run build**。
5. 添加必需的环境变量 `DATABASE_URL`。这些值在 [安装](/docs/install) 的 **配置 Umami** 步骤中有定义。
6. 部署并访问您的应用程序。
7. 按照从 [登录](/docs/login) 步骤开始的 **入门指南** 操作，务必更改默认密码。
