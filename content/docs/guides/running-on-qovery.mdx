---
title: 在 Qovery 上运行
---

[Qovery](https://www.qovery.com) 是一个完全托管的云平台，可以运行在您的 AWS、Digital Ocean 和 Scaleway 账户上，您可以在一个地方托管静态网站、后端 API、数据库、定时任务以及所有其他应用。

Qovery 为个人开发者提供**免费托管**，包括以下功能：

- 来自 GitHub 和 GitLab 的持续、自动构建与部署。
- 通过 [Let's Encrypt](https://letsencrypt.org) 自动获得 SSL 证书。
- 免费托管的 PostgreSQL 数据库。
- 免费 SSD 存储。
- 无限协作者。
- 无限 [自定义域名](https://hub.qovery.com/docs/using-qovery/configuration/application/#domains)。

## 设置

### 1. 创建 Qovery 账户

访问 [Qovery 控制台](https://start.qovery.com) 创建账户，如果您还没有的话。

### 2. 创建项目

- 点击 **创建项目** 并为您的项目命名。
- 点击 **下一步**。

![创建项目](https://hub.qovery.com/img/heroku/heroku-2.png)

### 3. 创建新环境

- 点击 **创建环境** 并命名（例如 staging、production）。

![创建新环境](https://hub.qovery.com/img/heroku/heroku-3.png)

### 4. 添加您的 Umami 应用

- 点击 **创建应用**，输入名称，选择包含您 Umami 应用的 GitHub 或 GitLab 仓库。
- 定义主分支名称和应用根路径。
- 点击 **创建**。

![添加应用](https://hub.qovery.com/img/rust/rust.png)

应用创建后：

- 进入您的应用 **设置**。
- 选择 **端口**。
- 添加您的 Umami 应用使用的端口。

### 5. 部署数据库

按照[本指南](https://hub.qovery.com/guides/getting-started/create-a-database)创建并部署一个新的 PostgreSQL 数据库。

### 6. 添加存储

要添加存储，进入应用 **设置**：

![添加存储](https://hub.qovery.com/img/add-storage.png)

### 7. 配置 Umami 应用

若使用 Qovery 提供的 PostgreSQL，您可以使用内置的 secrets 和环境变量。您可以在我们的[配置章节](https://hub.qovery.com/docs/using-qovery/configuration/environment-variable/)了解更多关于环境变量和 secrets 的信息。

### 8. 在 Qovery 上部署应用

现在只需进入您的应用并点击 **部署**。

![部署应用](https://hub.qovery.com/img/heroku/heroku-1.png)

就是这样。观察状态，等待应用部署成功。

要在浏览器中打开应用，点击应用概览中的 **操作**，然后点击 **打开**。
