## 设计模式

打开看板并点击 **Design** 按钮进入设计模式。在设计模式下，您可以添加行、调整列并放置组件。

![image](/images/docs/board-design.png)

## 添加行和列

点击 **Add row** 按钮插入新行。每行最多可容纳 **4 columns**。点击行内的 **Add column** 按钮可进一步拆分。

![image](/images/docs/board-add-row.png)

## 添加组件

点击 **Select Component** 按钮打开组件选择器。选择组件类型并配置其设置，然后点击 **Save** 进行放置。

![image](/images/docs/board-component.png)

## 组件

| Component | Category | Description | Board types |
| --- | --- | --- | --- |
| 网站指标栏 | 概览 | 显示浏览量、访客数、访问量、跳出率和站内停留时间的汇总栏。 | All |
| 网站图表 | 概览 | 页面浏览量和访客随时间变化的折线图。 | All |
| 指标表格 | 表格 | 可按维度（页面、引荐来源、浏览器、国家等）配置的指标表，支持调整行数限制。 | All |
| 世界地图 | 可视化 | 在世界地图上展示访客的地理分布。 | All |
| 每周流量 | 可视化 | 按星期几和小时显示流量强度的热力图。 | All |
| 事件图表 | 可视化 | 自定义事件随时间变化的柱状图。 | Website only |
| 文本块 | 内容 | 用于标签、注释或描述的自由文本。 | All |

## 调整行和列大小

可以通过拖动行和列之间的分隔线来调整行高和列宽。
