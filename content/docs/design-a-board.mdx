---
title: Design a board
---

*Available since v3.1.0*

## Design mode

Open a board and click the **Design** button to enter design mode. In design mode you can add rows, adjust columns, and place components.

![image](/images/docs/board-design.png)

## Add rows and columns

Click the **Add row** button to insert a new row. Each row can hold up to **4 columns**. Click the **Add column** button within a row to split it further.

![image](/images/docs/board-add-row.png)

## Add components

Click the **Select Component** button to open the component picker. Select a component type and configure its settings, then click **Save** to place it.

![image](/images/docs/board-component.png)

## Components

| Component | Category | Description | Board types |
| --- | --- | --- | --- |
| Website Metrics Bar | Overview | Summary bar showing views, visitors, visits, bounces, and time on site. | All |
| Website Chart | Overview | Line chart of page views and visitors over time. | All |
| Metrics Table | Tables | Configurable table of metrics by dimension (pages, referrers, browsers, countries, etc.) with adjustable row limits. | All |
| World Map | Visualization | Geographic distribution of visitors plotted on a world map. | All |
| Weekly Traffic | Visualization | Heatmap of traffic intensity by day of week and hour. | All |
| Events Chart | Visualization | Bar chart of custom events over time. | Website only |
| Text Block | Content | Free-form text for labels, annotations, or descriptions. | All |

## Resize rows and columns

Row heights and column widths are resizable by dragging the dividers between them.
