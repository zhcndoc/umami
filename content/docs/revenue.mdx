---
title: 收入
---

Umami 收入功能帮助您跟踪财务表现、衡量投资回报率（ROI），并识别哪些页面或产品产生了最多收入。它提供了对客户行为的洞察，帮助优化收入来源并提高转化率。通过将收入数据与流量和用户行为关联，企业能够做出明智决策，提升盈利能力并规划未来增长。

该洞察通过汇总**收入**和**货币**数据，覆盖指定的时间段来实现。

## 配置收入

要开始收集收入数据，您需要利用事件追踪并传递动态数据。这可以通过[追踪器函数](https://umami.is/docs/tracker-functions#event-data)或[数据属性](https://umami.is/docs/track-events#using-data-attributes)实现。被追踪的事件必须携带两个动态数据属性 `revenue` 和 `currency`。如果货币代码（ISO 4217）未被识别，洞察将默认使用 `USD`。

### 追踪器函数

```js
umami.track('checkout-cart', { revenue: 19.99, currency: 'USD' });
```

追踪事件时，默认属性包含在负载中。此代码相当于执行：

```js
umami.track(props => ({
  ...props,
  name: 'checkout-cart',
  data: {
    revenue: 19.99,
    currency: 'USD',
  },
}));
```

### 数据属性

请注意，使用此方式时，收入必须作为字符串传递。

```html
<button
  id="signup-button"
  data-umami-event="checkout-cart"
  data-umami-event-revenue="19.99"
  data-umami-event-currency="USD"
>
  结账
</button>
```

## 参数

- `Currency`：（必填）汇总数据所使用的货币。
