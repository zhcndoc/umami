---
title: 获取更新
---

## 源代码

为了获取最新的更新，首先从 Git 仓库拉取更改：

```shell
git pull
```

然后安装任何新的或更新的依赖项：

```shell
pnpm install
```

重新构建项目：

```shell
pnpm build
```

最后，启动应用：

```shell
pnpm start
```

## Docker

对于 Docker，只需拉取最新镜像；带有 PostgreSQL 支持：

```shell
docker pull docker.umami.is/umami-software/umami:latest
```
