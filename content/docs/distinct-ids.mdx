---
title: 唯一标识符
---

唯一标识符（Distinct ID）是分配给用户的唯一标识，无论是匿名还是登录状态。
这使你能够将多个会话中的操作（页面浏览、点击、转化）关联到同一个用户。
若想获得更全面的会话识别方式，请查看 [会话数据](/docs/tracker-functions#session-data)。

## 使用场景

- 准确归因用户行为。
- 连接多个会话或设备中的事件。
- 识别回访用户与新用户。
- 基于用户历史构建旅程和用户画像。

## 使用方法

在数据负载中添加 **id** 属性。该属性最多支持 50 个字符。

```json
{
  "payload": {
    "hostname": "your-hostname",
    "language": "en-US",
    "referrer": "",
    "screen": "1920x1080",
    "title": "dashboard",
    "url": "/",
    "website": "your-website-id",
    "id": "bob@aol.com"
  },
  "type": "event"
}
```
