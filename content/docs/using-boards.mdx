---
title: 使用看板
---

看板是可自定义的仪表盘，允许你在灵活的网格布局中排列分析组件。
你可以创建限定于特定网站、像素或链接的看板，也可以在单个视图中混合来自多个来源的组件。
看板可以与团队成员共享，也可以通过共享链接设为公开。

{/* ![image](/images/docs/boards-overview.png) */}

## 看板类型

| 类型 | 描述 |
| --- | --- |
| `mixed` | 无需必需实体的开放看板。可混合来自多个网站、像素和链接的组件。 |
| `website` | 作用域限定为特定网站。解锁仅适用于网站的组件。 |
| `pixel` | 作用域限定为特定像素。显示该像素收集的指标。 |
| `link` | 作用域限定为特定链接。显示该链接的点击和流量指标。 |
