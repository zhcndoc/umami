---
title: Using boards
---

Boards are customizable dashboards that let you arrange analytics widgets in a flexible grid layout.
You can create boards scoped to a specific website, pixel, or link, or mix components from multiple sources in a single view.
Boards can be shared with team members or made public via a share link.

## Board types

| Type | Description |
| --- | --- |
| `mixed` | Open board with no required entity. Mix components from multiple websites, pixels, and links. |
| `website` | Scoped to a specific website. Unlocks website-only components. |
| `pixel` | Scoped to a specific pixel. Displays metrics collected by that pixel. |
| `link` | Scoped to a specific link. Displays click and traffic metrics for that link. |

## Viewing a board

Click on a board name from the **Boards** list to open it. You will see all the components arranged in the layout defined during design. Use the date range selector to change the time period for all components on the board.

## Managing boards

- To create a new board, see [Add a board](/docs/add-a-board).
- To customize the layout and add components, see [Design a board](/docs/design-a-board).
- To share a board publicly, see [Add a board - Share a board](/docs/add-a-board#share-a-board).
