---
title: 使用团队
---

Umami 团队允许组织在共享的工作区下对网站、用户和资源进行分组，便于管理和协作。这种结构提升了组织性、访问控制和可见性。

## 角色

团队角色及相关权限。

| Role | Permissions |
| --- | --- |
| 团队所有者 | 所有权限。 |
| 团队管理者 | 除删除团队外的所有权限。 |
| 团队成员 | 除删除或更新团队外的所有权限。 |
| 仅查看 | 仅可查看。 |

## 团队访问

访问团队有两种方法。

**方法一：** 进入 **Teams**，然后点击 **名称** 列下的团队名称。

![image](/images/docs/team-view.png)

**方法二：** 使用 Umami 导航栏中的快速团队访问下拉菜单。

![image](/images/docs/team-quick-access.png)
