---
title: 导出数据
---

Umami 允许你将 Umami Cloud 中的数据导出为 CSV 格式（通过 gzip 压缩）。导出操作一次仅支持单个网站。
如果你有大量网站或大量数据需要导出，请联系 [support@umami.is](mailto:support@umami.is)。

## 从 Umami Cloud 导出数据

登录后，点击侧边导航栏的下拉菜单，然后点击 **设置**。

![image](/images/cloud/navbar-settings.png)

导航至 **数据**，然后点击 **导出** 按钮。

![image](/images/cloud/export-data.png)

填写表单信息后，点击 **导出** 按钮。

![image](/images/cloud/export-form.png)

导出完成后，你将收到一封包含导出下载链接的电子邮件。

![image](/images/cloud/export-email.png)
