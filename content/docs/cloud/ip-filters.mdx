---
title: IP 过滤器
---

Umami 允许屏蔽特定的 IP 地址或 IP 范围，这样您可以过滤传入流量并防止不受欢迎或恶意的连接。IP 过滤器功能从 **Pro** 计划开始提供。

## 添加 IP 过滤器

登录 Umami，点击侧边导航下拉菜单，然后点击 **设置**。

![image](/images/cloud/navbar-settings.png)

导航到 **IP 过滤器**，然后点击 **添加 IP 过滤器** 按钮。

![image](/images/cloud/ip-filter-add.png)

填写表单详情，然后点击 **保存** 按钮。

## 编辑或删除 IP 过滤器

导航到 **IP 过滤器**，点击您想要操作的 IP 过滤器旁的 **编辑** 按钮或 **删除** 按钮。

![image](/images/cloud/ip-filter-table.png)
