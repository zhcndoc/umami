# Umami Cloud 常见问题解答

### Umami Cloud 有免费套餐吗？

有的！Umami Cloud 的 Hobby 计划是完全免费的。非常适合个人项目和低流量网站。

### 免费试用是如何运作的？

Umami Cloud 的免费试用期自购买之日起持续14天。试用期间您可以完全访问该计划的所有功能。
试用结束后，您将按正常费用计费。您可以在试用期内任何时间取消试用，且无需支付费用。

### 我将如何被收费？

账单按月或按年结算。
如果您通过试用升级到付费计划，您的第一张发票将在试用期结束后生成。

### 使用量如何测量？

使用量通过计算网站访问次数以及任何自定义事件或自定义事件数据的存储情况来衡量。每次网站访问计为一个事件。如果您保存事件数据，存储的每个数据属性都计为一个事件。

### 你们的服务器位于哪里？

Umami Cloud 的服务器位于美国和欧盟，遵守 GDPR 和 CCPA 法规。

### 我可以导出我的数据吗？

可以，您所有的数据都可以从 Umami Cloud 导出。
