---
title: 白标
---

**白标** 允许你使用自己的 Logo、显示名称和链接来自定义 Umami 分享页面上的品牌标识。白标功能仅适用于 **Business** 方案。

当你与客户或利益相关者分享分析仪表盘时，这个功能非常有用——访客看到的不是 Umami 的品牌，而是你公司的名称和 Logo。无论你是向客户展示数据的代理机构，还是分享面向公众的分析页面的企业，它都能让你的报表看起来更专业，并保持品牌一致性。

![image](/images/cloud/white-label-details.png)

## 配置白标

登录 Umami，然后点击侧边导航中的个人资料按钮，再点击 **设置**。

![image](/images/cloud/navbar-settings.png)

导航到 **White-label** 并填写表单详情。

![image](/images/cloud/white-label-form.png)

| Field | Description |
| --- | --- |
| Logo | 上传一张图片，显示在你的分享页面上。 |
| Display Name | 显示在你的分享页面上的品牌名称。 |
| Link URL | 访客点击你的 Logo 或显示名称时会跳转到的 URL。 |

点击 **保存** 按钮以应用你的更改。
