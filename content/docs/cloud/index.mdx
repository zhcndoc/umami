Umami Cloud 是由 Umami 创建者运营的 Umami 应用托管版本。

## 优势

- **完全托管**：无需自行托管和管理服务器，即可获得所有最新更新。
- **可扩展**：Umami 提供无缝扩展，允许您处理不断增长的数据量和用户需求，而不影响性能。
- **成本高效**：采用简单的按使用量计费模式，只需为实际使用付费。
- **合规**：符合 GDPR 和 CCPA 规定。
