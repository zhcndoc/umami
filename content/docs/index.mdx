Umami 是一个尊重用户隐私的开源网络分析平台。无 Cookie，无跨站点追踪，无个人数据收集。开箱即用，符合 GDPR 要求。

本文档涵盖 **Umami v3**。前一版本请参见 [v2 文档](https://v2.umami.is)。

## 为什么选择 Umami？

- **隐私优先** — 无 Cookie，无指纹识别，无个人数据。您的访客保持匿名。
- **拥有您的数据** — 在您的基础设施上自行托管。您的分析数据永远不会离开您的服务器。
- **轻量级** — 追踪脚本小于 2KB。不会拖慢您的网站速度。
- **简单** — 使用 Docker 几分钟即可部署。不需要复杂配置。

## 快速开始

```bash
docker compose up -d
```

就是这么简单。打开 `http://localhost:3000`，使用 `admin` / `umami` 登录。

有关详细的安装选项，请参见 [安装](/docs/install)。如需托管平台指南，请查看我们的 [部署指南](/docs/guides/hosting)。

## 功能

**核心分析** — 页面浏览量、访客数、跳出率、会话时长、推荐来源、浏览器、操作系统、设备和国家统计。

**自定义事件** — 通过简单的数据属性或 JavaScript 调用跟踪按钮点击、表单提交或任何用户交互。

**高级洞察** — 漏斗分析、用户路径、留存分析、目标设定、UTM 活动跟踪以及群组细分。

**会话** — 查看单个访客活动和会话属性，无需识别个人信息。

**团队** — 与团队协作。共享网站，分配角色并管理访问权限。

**API** — 完整的 REST API，实现访问所有分析数据的编程接口。
