title: 常见问题
---

### 1. Umami 是否符合 GDPR 规定？

是的，Umami 不会收集任何可识别个人身份的信息，并对所有收集的数据进行匿名处理。用户无法被识别，也不会被跨网站追踪。

### 2. 我需要向用户展示 Cookie 通知吗？

不需要，Umami 的追踪代码中不使用任何 Cookie。

### 3. Umami 是否支持单页应用（SPA）？

支持，Umami 可以无缝兼容单页应用和普通网站。

### 4. Umami 能记录按钮点击等事件吗？

可以，请参见 **入门指南** 中的 [事件追踪](/docs/track-events)。

### 5. 如果我遇到问题或想请求新功能，应该去哪儿寻求帮助？

对于一般讨论和故障排查，可以使用 [GitHub 讨论区](https://github.com/umami-software/umami/discussions)。

对于错误报告和功能请求，请在 GitHub 上提交 [问题](https://github.com/umami-software/umami/issues)。

想要更多乐趣，可以加入 Umami 社区的 [Discord](https://discord.gg/4dz4zcXYrQ)！
