见解报告允许您通过高级过滤对网站和应用分析进行切片。过滤选项包括 URL、页面标题、引荐来源、查询、浏览器、操作系统、设备、区域、城市和主机。然后，见解报告将对该子集的数据进行浏览次数、访问次数、访客数、跳出率和访问时长的详细分析。见解报告的神奇之处在于，您可以使用多个过滤器来真正深入了解事物。见解报告使创始人和产品团队能够发现隐藏的模式，识别关键趋势，并做出数据驱动的决策，以改善绩效。

让我们探索创业公司如何利用 Umami 见解报告推动增长和优化用户体验的五种实际方法。

## 见解报告简介

在深入具体用例之前，让我们简要介绍一下如何使用见解报告：

1. 在您的 Umami 控制面板中，导航到报告部分并创建一个新的见解报告（见解报告在 Umami 的所有版本中都是标准配置）。
2. 选择您的网站和日期范围。
3. 选择您想要分析的字段（例如，URL、页面标题、引荐来源、操作系统、设备、国家）。
4. 运行报告。

有关设置和使用见解报告的详细说明，请参阅我们的 [文档](https://umami.is/docs/reports/report-insights)。

现在，让我们探索创业公司如何利用见解报告推动增长并做出数据驱动的决策的五种方法。

## 内容绩效分析

对于专注于内容营销或拥有内容丰富产品的创业公司，见解报告可以提供哪些内容与用户产生共鸣的数据。

使用 URL 或页面标题字段按内容类型或特定页面对数据进行细分。应用过滤器，关注页面停留时间、跳出率或与内容参与度相关的页面浏览量或查询（可以捕捉 URL 中的 UTM 参数）。

例如，您可能会发现您的操作指南与公司新闻文章相比，跳出率明显较低，平均页面停留时间较长。这项见解可以为您的内容策略提供信息，促使您创建更多实用的内容，以保持用户的参与。

## 用户细分和行为分析

见解报告的细分功能使其非常适合分析不同用户组如何与您的产品互动。

结合设备、操作系统和国家等字段创建特定用户区块。然后，分析这些区块在参与度指标、功能使用或转化率方面的差异。

您可能会发现，来自特定国家的移动用户在某个特定功能上的转化率远高于其他用户。这一见解可能促使您优化该市场的移动体验或者创建针对该高效能用户区块的市场营销活动。

## 获取渠道优化

对于专注于增长的创业公司，了解哪些获取渠道最有效至关重要。见解报告可以帮助您深入挖掘流量来源及其质量。

使用引荐来源字段按流量来源对数据进行细分。结合跳出率、每次会话页面数和转化事件等指标来评估每个来源的流量质量。

您可能会发现，某个特定社交媒体平台驱动了大量流量，但来自该来源的访客跳出率高且转化率低。相反，您可能会发现，相对较低的自然搜索流量具有更高的参与度和转化率。这些见解可以帮助您更有效地分配营销资源。

## 特性采用和使用模式

对于以产品为主的创业公司，了解用户如何与不同功能互动对产品开发和用户体验优化至关重要。

分析产品中关键功能的 URL 或页面标题，然后使用见解报告分析不同用户区块如何与这些功能互动。您可以按设备、地理位置或任何其他相关维度进行细分。

例如，您可能会发现 Android 用户频繁使用某个协作功能，而 iOS 用户几乎不接触它。这一见解可以为您的产品路线图提供信息，促使您为 Android 用户开发更多协作工具，或者创建教育内容以提高 iOS 用户的采用率。

## 用户旅程和漏斗分析

尽管 Umami 提供专门的 [漏斗](https://umami.is/blog/optimizing-conversion-paths-using-the-funnel-report) 和 [旅程](https://umami.is/blog/perfect-your-understanding-of-user-journeys) 报告，见解报告可以为您理解用户在产品中的路径提供额外的深度。

使用 URL 字段分析用户通过您的网站或产品通常走的路径。应用过滤器，关注有价值的旅程，例如那些导致转化或关键功能使用的路径。

您可能会发现，在访问您的功能页面后直接访问定价页面的用户更可能注册试用。这一见解可能促使您优化这两个页面之间的路径，或在功能页面上增加更显著的号召性用语，以引导用户获取定价信息。
