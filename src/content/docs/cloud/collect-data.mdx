# 收集数据

要获取你的跟踪代码，请点击要跟踪的网站上的**编辑**按钮。

<img src="/images/cloud/website-edit.png" />

跟踪代码可以在**跟踪代码**选项卡下找到。

复制代码，并将其插入到你的网站的`<head>`部分。

<img src="/images/cloud/website-tracking-code.png" />

然后访问你的网站，数据应立即出现在你的奥马尼仪表板中。

### 故障排除

- 如果你的网站是使用 [Next.js](https://nextjs.org/) 构建的，请使用 [next/script](https://nextjs.org/docs/app/api-reference/components/script) 组件插入上述代码块。
