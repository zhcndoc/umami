# 启用共享网址

默认情况下，只有账户所有者才能查看网站的指标。但是您可以选择通过公共的**共享网址**或者通过**团队**来使统计数据可见。

## 共享网址

转到**网站**，点击网站旁边的**编辑**按钮。

![Edit Button](/images/cloud/website-edit.png)

转到**共享网址**选项卡，打开共享网址开关。

![Share URL](/images/cloud/share-url.png)

弹出的表单将显示您的 URL。

![Share URL Form](/images/cloud/share-url-form.png)

您可以通过点击**重新生成**按钮并保存来生成一个新的唯一共享网址哈希串。
