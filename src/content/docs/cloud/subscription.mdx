---
title: 订阅
---

Umami 使用 **Stripe** 作为其支付处理器。注册计划时，您将被带到 Stripe 结账门户。
所有订阅均附带 14 天的免费试用期。在此期间，您可以免费取消。

有关每个 Umami Cloud 计划的详细信息，请访问 [定价](https://umami.is/pricing) 页面。

## 升级到专业版

导航到 **账单** 并点击 **升级到专业版** 按钮。

![image](/images/cloud/billing-plans.png)

如果您对 **企业** 计划感兴趣，请点击 **联系我们** 按钮或发送电子邮件至 sales@umami.is

您将被带到 Stripe 结账门户以完成注册。导航到 **账单** 查看您的新账单详细信息。

![image](/images/cloud/billing-details.png)

## 降级计划

导航到 **账单** 并点击 **降级计划** 按钮。使用此选项取消您的 **专业** 计划。

- 如果您已经达到了每月爱好限制，您的网站将被阻止收集数据，直到下一个周期。
- 如果您超过了网站限制，我们将计算流量最多的前三个网站并显示这些网站。如果您升级回**专业**账户，您剩余的网站将被重新激活。

![image](/images/cloud/billing-downgrade-plan.png)

## 管理计划

导航到 **账单** 并点击 **管理账单信息** 按钮。您将被带到 Stripe 账单门户，在那里您可以执行以下操作。

- 取消您的 **专业** 计划
- 更新支付方式
- 更新账单信息
- 查看发票历史记录
- 下载发票和收据

![image](/images/cloud/billing-manage-plan.png)
