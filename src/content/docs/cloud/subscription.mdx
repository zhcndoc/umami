# 订阅

Umami 使用 **Stripe** 作为其支付处理器。当注册计划时，您将被发送到 Stripe 支付门户。
所有订阅都附带 14 天免费试用期。您可以在该时间范围内免费取消订阅。

有关每个 Umami 云计划的详情，请参阅 [定价](https://umami.is/pricing) 页。

## 升级到 Pro

转到 **计费** 并点击 **升级到 Pro** 按钮。

<img src="/images/cloud/billing-plans.png" />

选择您的计划和年度计费选项，然后点击 **结账** 按钮。

如果您对 **企业** 计划感兴趣，请点击 **联系我们** 按钮或发送电子邮件至 sales@umami.is

<img src="/images/cloud/billing-form.png" />

您将被发送到 Stripe 支付门户来完成注册。转到 **计费** 查看您的新计费详情。

<img src="/images/cloud/billing-details.png" />

## 更改计划

转到 **计费** 并点击 **更改计划** 按钮。使用此选项来更改您的 **Pro** 计划价格层或更改计费频率。

<img src="/images/cloud/billing-change-plan.png" />

## 降级计划

转到 **计费** 并点击 **降级计划** 按钮。使用此选项来取消您的 **Pro** 计划。

- 如果您已达到每月活动限制，您的网站将被阻止收集数据，直到下一个周期。
- 如果您超过网站限制，我们将计算出三个流量最大的网站并显示出来。如果您升级回到 **Pro** 账户，您剩下的网站将重新激活。

<img src="/images/cloud/billing-downgrade-plan.png" />

## 管理计划

转到 **计费** 并点击 **管理计费信息** 按钮。您将被发送到 Stripe 计费门户，您可以进行以下操作。

- 取消您的 **Pro** 计划
- 更新付款方式
- 更新计费信息
- 查看发票历史
- 下载发票和收据

<img src="/images/cloud/billing-manage-plan.png" />
