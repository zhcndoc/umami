# 导出数据

Umami 允许你将数据从 Umami 云导出为 CSV 格式（通过 gzip 进行压缩）。一次只能处理一个网站的导出。
如果你有大量的网站或大量的数据需要导出，请联系 [support@umami.is](mailto:support@umami.is)。

## 从 Umami 云导出数据

登录 Umami 并导航至 **账户**。点击 **数据** 标签，然后点击 **导出** 按钮。

<img src="/images/cloud/export-data.png" />

填写表单详情，然后点击 **导出** 按钮。

<img src="/images/cloud/export-form.png" />

导入完成后，你将收到一封电子邮件包含下载导出文件的链接。

<img src="/images/cloud/export-email.png" />
