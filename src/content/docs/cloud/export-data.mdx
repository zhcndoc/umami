---
title: 导出数据
---

Umami 允许您将数据从 Umami Cloud 导出为 CSV 格式（通过 gzip 压缩）。导出一次处理一个网站。如果您有大量网站或大量数据需要导出，请联系 [support@umami.is](mailto:support@umami.is)。

## 从 Umami Cloud 导出数据

登录 Umami 并导航至侧边栏下拉菜单，然后点击 **设置**。

![image](/images/cloud/navbar-settings.png)

导航至 **数据**，然后点击 **导出** 按钮。

![image](/images/cloud/export-data.png)

填写表单信息并点击 **导出** 按钮。

![image](/images/cloud/export-form.png)

完成导出后，您将收到一封电子邮件，包含下载导出的链接。

![image](/images/cloud/export-email.png)
