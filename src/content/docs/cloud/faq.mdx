# Umami Cloud 常见问题解答

### Umami Cloud 有免费套餐吗？

有的！Umami Cloud 的 Hobby 套餐是完全免费的，非常适合个人项目和低流量网站。

### 免费试用是如何工作的？

Umami Cloud 的免费试用期为自购买之日起的 14 天。试用期间，您可以完全访问该套餐的所有功能。
试用结束时，您将按常规金额被收取费用。在试用期间，您可以随时免费取消试用。

### 我将如何被收费？

收费可以按月或按年进行。
如果您通过试用升级到付费套餐，您的第一张发票将在试用期结束后发送。

### 如何衡量使用量？

使用量是通过计算对网站的访问量以及任何自定义事件或存储的自定义事件数据来衡量的。每次访问网站计为一个事件。如果您保存了事件数据，则每个存储的数据属性也计为一个事件。

### 你们的服务器位于哪里？

Umami Cloud 的服务器位于美国和欧盟，并遵守 GDPR 和 CCPA 的法规。

### 我可以导出我的数据吗？

可以，您所有的数据都可以从 Umami Cloud 导出。
