# Umami 云常见问题解答

### Umami 云有免费的套餐吗？

有！Umami 云的 Hobby 套餐是完全免费的。非常适合个人项目和低流量网站。

### 免费试用如何工作？

Umami 云的免费试用将从购买日期开始持续 14 天。试用期内，您可以完全访问该套餐的所有功能。试用期结束时，您将被收取常规金额。您可以在试用期内随时取消试用，不会产生任何费用。

### 我将如何收费？

计费方式分为按月和按年计费。如果您通过试用升级到付费套餐，则您的第一张发票将在试用期结束后到达。

### 如何计量使用量？

使用量是通过统计访问网站的次数以及存储的任何自定义事件或自定义事件数据来计算的。每个访问网站的次数被视为一个事件。如果保存事件数据，每个存储的数据属性都被视为一个事件。

### 您的服务器位于哪里？

Umami 云服务器位于美国和欧洲，并遵守 GDPR 和 CCPA 规定。

### 我可以导出我的数据吗？

是的，您可以从 Umami 云中导出所有数据。
