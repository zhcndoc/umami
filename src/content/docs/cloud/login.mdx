# 登录

你可以在 [https://cloud.umami.is/login](https://cloud.umami.is/login?ref=umami-cloud-docs) 上登录到你的账户。

<img src="/images/cloud/login-form.png" />

如果你忘记了密码，点击 **忘记密码？** 按钮并输入你的电子邮箱，你将收到一封重置密码的邮件。

<img src="/images/cloud/login-forgot-password.png" />

你可以通过导航到 **账户** 并点击 **更改密码** 按钮来修改你的密码。

<img src="/images/cloud/login-change-password.png" />
