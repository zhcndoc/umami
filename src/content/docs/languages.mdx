Umami 支持显示不同的语言。只需点击标题中的地球图标即可更改语言。

![image](/images/docs/v2/languages.png)

## 贡献

如果您想为项目贡献语言翻译，请遵循以下步骤。

**1. 检出代码库**

```shell
git clone https://github.com/umami-software/umami.git
```

**2. 创建一个语言 `.json` 文件**

在 `/lang` 文件夹中应该有几个 `.json` 文件。将 `en-US.json` 文件复制到另一个文件中，并使用适当的 ISO 语言代码命名。

**3. 翻译 `.json` 文件中的字符串**

例如，以下是原始的英语翻译， [en-US.json](https://raw.githubusercontent.com/umami-software/umami/master/src/lang/en-US.json)。

这是其翻译成德语的版本，[de-DE.json](https://raw.githubusercontent.com/umami-software/umami/master/src/lang/de-DE.json)。

**4. 最后，在 [GitHub](https://github.com/umami-software/umami) 提交一个拉取请求**

如果您不熟悉使用 Git，可以在 GitHub 上创建一个 [issue](https://github.com/umami-software/umami/issues)，并粘贴您的 `.json` 文件内容，其他人会合并它。
