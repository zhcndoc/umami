Umami 支持显示不同语言。只需点击标题中的地球图标即可更改您的语言。

![image](/images/docs/v2/languages.png)

## 贡献

如果你想为项目做出语言翻译的贡献，请按照以下步骤操作。

**1. 检出代码库**

```shell
git clone https://github.com/umami-software/umami.git
```

**2. 创建一个语言 `.json` 文件**

在 `/lang` 文件夹中应该有几个 `.json` 文件。将 `en-US.json` 文件复制到另一个以正确的 ISO 语言代码命名的文件中。

**3. 翻译 `.json` 文件中的字符串**

例如，这是原始的英文翻译，[en-US.json](https://raw.githubusercontent.com/umami-software/umami/master/src/lang/en-US.json)。

这是将其翻译成德文后的文件，[de-DE.json](https://raw.githubusercontent.com/umami-software/umami/master/src/lang/de-DE.json)。

**4. 最后，在 [GitHub](https://github.com/umami-software/umami) 上提交拉取请求**

如果你不熟悉使用 Git，你可以在 GitHub 上简单地创建一个 [issue](https://github.com/umami-software/umami/issues)，然后粘贴 `.json` 文件的内容，有人会合并它。
