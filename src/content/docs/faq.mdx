title: 常见问题解答
---

### 1. Umami 符合 GDPR 吗？

是的，Umami 不收集任何可识别个人身份的信息，并对所有收集的数据进行匿名处理。用户无法被识别，并且从未在各个网站上被跟踪。

### 2. 我需要向用户显示 cookie 通知吗？

不需要，Umami 在跟踪代码中不使用任何 cookies。

### 3. Umami 可以在单页应用（SPA）上工作吗？

可以，Umami 在 SPA 和普通网站之间无缝工作。

### 4. Umami 能记录按钮点击等事件吗？

可以，请参阅 **入门指南** 下的 [跟踪事件](/docs/track-events)。

### 5. 如果我遇到问题或想请求某个功能，我可以去哪里寻求帮助？

对于一般讨论和故障排除，您可以使用 [GitHub 讨论](https://github.com/umami-software/umami/discussions)。

对于错误报告和功能请求，请在 GitHub 上提交 [问题](https://github.com/umami-software/umami/issues)。

想要更有趣的体验，可以加入 Umami 社区，在 [Discord](https://discord.gg/4dz4zcXYrQ) 等待您！
