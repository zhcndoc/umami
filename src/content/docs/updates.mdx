---
title: 获取更新
---

## 源代码

为了获取最新更新，首先从 Git 仓库拉取更改：

```shell
git pull
```

然后安装任何新的或更新的依赖项：

```shell
yarn install
```

重建项目：

```shell
yarn build
```

最后，启动应用程序：

```shell
yarn start
```

## Docker

对于 Docker，只需拉取最新的数据库配置镜像。

PostgreSQL：

```shell
docker pull docker.umami.is/umami-software/umami:postgresql-latest
```

或者 MySQL：

```shell
docker pull docker.umami.is/umami-software/umami:mysql-latest
```
