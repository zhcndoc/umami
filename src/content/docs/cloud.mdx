Umami Cloud 是由 Umami 创作者运营的 Umami 应用程序的托管版本。

## 好处

- **完全托管**: 无需自行托管和管理服务器，即可获得所有最新更新。
- **可扩展**: Umami 提供无缝的可扩展性，允许你处理不断增长的数据量和用户需求，而不会影响性能。
- **成本高效**: 使用我们简单的按使用量计费模型，只需为实际使用付费。
- **合规**: 符合 GDPR 和 CCPA 法规。

## 注册

你可以在 [https://cloud.umami.is/signup](https://cloud.umami.is/signup?ref=umami-cloud-docs) 免费注册。

有关每个 Umami Cloud 计划的详细信息，请参阅 [定价](https://umami.zhcndoc.com/pricing) 页面。
