---
title: 跟踪器功能
---

Umami 跟踪器提供了一些函数，如果您希望更好地控制跟踪，可以在网站上调用这些函数。

### umami.trackEvent(_事件名称_, [*事件数据*], [*网址*], [*网站 ID*])

跟踪具有给定名称的事件。

用法：

```js
umami.trackEvent(
  '注册按钮点击',
  { type: 'signup', userId: 123 },
  '/home',
  '94db1cb1-74f4-4a40-ad6c-962362670409',
);
```

您可以选择将 **事件数据** 作为 JSON 对象传递。

**网址** 和 **网站 ID** 的值是可选的。它们将默认为页面网址和由脚本定义的 `data-website-id`。

### umami.trackView(_网址_, [*引荐来源*], [*网站 ID*])

跟踪页面查看。

用法：

```js
umami.trackView('/home', 'https://www.google.com', '94db1cb1-74f4-4a40-ad6c-962362670409');
```

**引荐来源** 和 **网站 ID** 的值是可选的。它们将默认为页面引荐来源和由脚本定义的 `data-website-id`。
