---
title: 在 Qovery 上运行
---

[Qovery](https://www.qovery.com) 是一个完全托管的云平台，运行在你的 AWS、Digital Ocean 和 Scaleway 账户上，你可以在一个地方托管静态网站、后端 API、数据库、定时任务以及所有其他应用程序。

Qovery 为个人开发者提供 **免费托管**，包括以下功能：

- 从 GitHub 和 GitLab 进行持续的自动构建和部署。
- 通过 [Let's Encrypt](https://letsencrypt.org) 自动生成 SSL 证书。
- 免费托管 PostgreSQL。
- 免费 SSD 存储。
- 无限合作者。
- 无限 [自定义域名](https://hub.qovery.com/docs/using-qovery/configuration/application/#domains)。

## 设置

### 1. 创建 Qovery 账户

访问 [Qovery 仪表板](https://start.qovery.com) 创建账户，如果你还没有账户的话。

### 2. 创建项目

- 点击 **创建项目**，为你的项目命名。
- 点击 **下一步**。

![创建项目](https://hub.qovery.com/img/heroku/heroku-2.png)

### 3. 创建新环境

- 点击 **创建环境** 并命名（例如：staging, production）。

![创建新环境](https://hub.qovery.com/img/heroku/heroku-3.png)

### 4. 添加你的 Umami 应用

- 点击 **创建应用**，给应用命名并选择你的 GitHub 或 GitLab 仓库，其中包含你的 Umami 应用。
- 定义主分支名称和根应用路径。
- 点击 **创建**。

![添加应用](https://hub.qovery.com/img/rust/rust.png)

应用创建后：

- 访问你的应用 **设置**。
- 选择 **端口**。
- 添加 Umami 应用使用的端口。

### 5. 部署数据库

通过 [遵循此指南](https://hub.qovery.com/guides/getting-started/create-a-database) 创建并部署新的 PostgreSQL 数据库。

### 6. 添加存储

要添加存储，请访问你的应用 **设置**：

![添加存储](https://hub.qovery.com/img/add-storage.png)

### 7. 设置 Umami 配置

要使用 Qovery 提供的 PostgreSQL，你可以使用内置的秘密和环境变量。你可以在我们的 [配置部分](https://hub.qovery.com/docs/using-qovery/configuration/environment-variable/) 阅读更多关于环境变量和秘密的信息。

### 8. 在 Qovery 上部署应用

现在你只需访问你的应用，然后点击 **部署**。

![部署应用](https://hub.qovery.com/img/heroku/heroku-1.png)

就这样。查看状态并等待应用部署完成。

要在浏览器中打开应用，点击应用概览中的 **操作** 和 **打开**。
