title: 常见问题解答
---

### 1. Umami 符合 GDPR 吗？

是的，Umami 不收集任何可识别个人身份的信息，并对收集的所有数据进行匿名处理。用户无法被识别，并且在网站之间从不被追踪。

### 2. 我需要向用户显示 Cookie 通知吗？

不需要，Umami 的追踪代码中不使用任何 Cookies。

### 3. Umami 是否适用于单页应用程序（SPA）？

是的，Umami 在 SPA 和普通网站之间无缝工作。

### 4. Umami 能否记录按钮点击等事件？

可以，详情请参见 **快速入门** 下的 [跟踪事件](/docs/track-events)。

### 5. 如果我遇到问题或想请求一个功能，我应该去哪里寻求帮助？

对于一般讨论和故障排除，您可以使用 [GitHub Discussions](https://github.com/umami-software/umami/discussions)。

对于错误报告和功能请求，请在 GitHub 上提交一个 [问题](https://github.com/umami-software/umami/issues)。

想要体验更多乐趣，可以加入 Umami 社区的 [Discord](https://discord.gg/4dz4zcXYrQ)!
