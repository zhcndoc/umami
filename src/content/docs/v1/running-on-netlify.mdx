---
title: 在 Netlify 上运行
---

[Netlify](https://netlify.com/) 提供了一个免费的托管服务，非常适合 GitHub 组织使用。

在此设置中，您应该已经有一个数据库服务器正在运行并接受远程连接。如果您还没有数据库，可以按照 [在 DigitalOcean 上运行](/docs/running-on-digitalocean) 的指南来启动一个数据库。您还可以查看 [托管](/docs/guides/hosting) 下的 **托管数据库** 部分。

## 设置 [![与 netlify 部署](https://www.netlify.com/img/deploy/button.svg)](https://app.netlify.com/start/deploy?repository=https://github.com/umami-software/umami)

_使用上面的按钮自动化步骤 1-5_

1. 将 [https://github.com/umami-software/umami](https://github.com/umami-software/umami) 项目分叉到您的 GitHub 账户。
2. 在 [Netlify](https://netlify.com/) 上创建一个账户。
3. 从仪表板页面点击 **导入项目**，然后指定您在 GitHub 上的项目分叉的 URL。
4. 将构建命令更改为 **yarn run build**。
5. 添加所需的环境变量 `DATABASE_URL`。这些值在 [安装](/docs/install) 的 **配置 Umami** 步骤中定义。
6. 部署并访问您的应用程序。
7. 从 [登录](/docs/login) 步骤开始，遵循 **入门指南**，并确保更改默认密码。
