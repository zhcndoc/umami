---
title: 在 Supabase 上运行
---

[Supabase](https://supabase.com/) 是一个基于 Postgres 数据库的开源 Firebase 替代品。

## 设置

1. 在 Supabase 上创建一个项目，选择一个接近您将托管 Umami 项目的区域的名称。
2. 从 **设置 > 数据库** 页面获取数据库连接字符串，然后滚动到底部找到 **连接池** 部分并复制 **连接字符串**。它应该类似于：`postgres://postgres:[YOUR-PASSWORD]@host:6543/postgres`
3. **重要：** 在您刚复制的连接字符串中添加 `?pgbouncer=true`。
4. 将 `DATABASE_URL` 添加到您的 `.env` 文件：

   ```
   DATABASE_URL=postgres://postgres:[YOUR-PASSWORD]@host:6543/postgres?pgbouncer=true
   ```

5. 现在您应该能够构建并启动 Umami (`npm run build` 然后 `npm start`)。
6. 请按照 **入门指南** 从 [登录](/docs/login) 步骤开始，并确保更改默认密码。
