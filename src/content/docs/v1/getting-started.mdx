---
title: 开始使用 Umami
---

Umami 是一个自托管的网站分析解决方案。运行 Umami 的方式有很多种。

如果您想直接从源代码安装或构建自己的 Docker 镜像，可以直接跳到 [安装](/docs/install) 部分。

## Docker

如果您想要预构建的 Docker 镜像用于您的数据库，可以直接从我们的注册中心拉取：

### PostgreSQL

```shell
docker pull docker.umami.is/umami-software/umami:postgresql-latest
```

### MySQL

```shell
docker pull docker.umami.is/umami-software/umami:mysql-latest
```

## 托管

如果您不熟悉运行自己的服务器，请查看 [托管](/docs/guides/hosting) 部分以获得快速概述。

还有许多服务可以运行 Umami。请参阅导航菜单中的 **指南** 部分。

快速启动的一种方式是使用 Railway。

[![在 Railway 上部署](https://railway.app/button.svg)](https://railway.app/new/template/umami-analytics)

请参阅 [在 Railway 上运行](/docs/running-on-railway) 以开始使用。
