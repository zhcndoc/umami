# 报告

Umami 内置了报告功能，帮助您更深入地了解您的数据。
这些报告共同提供了对用户行为、满意度和优化机会的全面理解。

## 可用的报告

- [洞察](/docs/reports/report-insights) - 通过使用细分和过滤器来更深入地了解您的数据。
- [漏斗](/docs/reports/report-funnel) - 了解用户的转化率和流失率。
- [留存](/docs/reports/report-retention) - 通过跟踪用户的回访频率来衡量您的网站粘性。
- [UTM](/docs/reports/report-utm) - 通过 UTM 参数追踪您的广告活动。

## 创建报告

在单击网站上的**查看**后，即可访问报告。

导航到**报告**，然后单击**创建报告**按钮。

<img src="/images/v2/report-create.png" />

## 访问报告

您可以通过单击各自的**报告**按钮来访问所有报告或特定网站的报告。

导航到**报告**，然后单击**查看**按钮。

<img src="/images/v2/report-detail.png" />
