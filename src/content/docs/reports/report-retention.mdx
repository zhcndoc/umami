# 用户留存

Umami 用户留存通过追踪用户多频繁返回来衡量您的网站的黏性。
该报告提供了有关有多少用户在长期内与您的网站保持互动的洞察，帮助您了解和提高用户的长期忠诚度。

报告使用了一种队列分析图表，为不同用户队列的长期留存趋势和性能提供了宝贵的洞察。
该图表按照每天不同的访客和这些访客在首次访问之后的特定天数内多频繁返回进行了分解。

## 参数

- `网站`：（必填）您要运行报告的网站。
- `日期`：（必填）报告的所选月份和年份。
