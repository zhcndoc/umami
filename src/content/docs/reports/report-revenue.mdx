---
title: 收入报告
---

Umami 收入功能帮助您跟踪财务表现、衡量投资回报率，并识别哪些页面或产品产生最多收入。
它提供了客户行为的洞察，帮助优化收入来源并提高转化率。
通过将收入数据与流量和用户行为关联起来，企业可以做出明智的决策，
以提高盈利能力并规划未来的增长。

该报告通过聚合指定时间范围内的 **收入** 和 **货币** 数据来工作。

## 配置收入

要开始收集收入数据，您需要利用动态数据跟踪事件。
这可以通过 [跟踪器函数](https://umami.is/docs/tracker-functions#event-data) 或 [数据属性](https://umami.is/docs/track-events#using-data-attributes) 来实现。
被跟踪的事件必须与两个动态数据属性 `revenue` 和 `currency` 一起传递。如果未识别货币代码（ISO 4217），报告将默认使用 `USD`。

### 跟踪器函数

```js
umami.track('checkout-cart', { revenue: 19.99, currency: 'USD' });
```

在跟踪事件时，默认属性会包含在负载中。
这相当于运行：

```js
umami.track(props => ({
  ...props,
  name: 'checkout-cart',
  data: {
    revenue: 19.99,
    currency: 'USD',
  },
}));
```

### 数据属性

请注意，使用此方法时，收入必须以字符串形式传递。

```html
<button
  id="signup-button"
  data-umami-event="checkout-cart"
  data-umami-event-revenue="19.99"
  data-umami-event-currency="USD"
>
  结账
</button>
```

## 参数

- `网站`: (必填) 您想运行报告的网站。
- `日期范围`: (必填) 报告的日期范围。
- `货币`: (必填) 要聚合的数据的货币。
