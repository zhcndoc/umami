# UTM

Umami UTM 通过跟踪不同营销渠道（如社交媒体、电子邮件和广告）的有效性，提供关于营销活动表现的分析数据。

该报告通过将 5 个标准 UTM 参数（来源、媒介、活动、关键词和内容）进行拆分，得到**视图**。

## 参数

- `日期范围`：（必填）报告的日期范围。
