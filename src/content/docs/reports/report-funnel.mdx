# 漏斗

Umami 漏斗帮助您了解用户的转化和流失率。
通过分析漏斗，用户可以做出明智的决策，优化网站的用户体验，实现更好的转化率。

该报告通过显示到达每个URL或事件的用户数量和与前一步的流失率来工作。
漏斗使用特定的顺序条件，要求按照所要求的顺序完成步骤。
用户可以在漏斗步骤之间导航到其他URL或事件，但必须按照顺序完成漏斗步骤才能被计算为转化。

## 参数

- `网站`：（必填）您想要运行报告的网站。
- `日期范围`：（必填）报告的日期范围。
- `窗口`：（必填）在漏斗步骤之间用户有多少分钟可以被计算为转化。
- `步骤`：（必填）漏斗中使用的连续步骤条件。至少需要两个步骤才能运行报告。

### 第3步：选择一个窗口（分钟）

<img src="/images/v2/funnel-window.png" />

### 第4步：选择您的步骤

点击**加号**，选择步骤类型（URL 或事件），在漏斗上输入第一个步骤条件，然后点击**添加**按钮。

<img src="/images/v2/funnel-url-form.png" />

继续添加步骤，直到完成您所需的漏斗条件。Umami 支持特定 URL、事件和URL通配符作为步骤。

<img src="/images/v2/funnel-url-details.png" />

- **特定 URL**：用户必须到达这个特定 URL。（例如：`/`）
- **事件**：用户必须生成这个特定的事件。（例如：`live-demo-button`）
- **URL通配符**：用户必须到达符合通配符条件的任何URL。只有`以...结束`的搜索可用。（例如：`/docs*`）

### 第5步：运行查询并保存

点击**运行查询**以查看结果。为了保存您的报告，添加一个标题和描述，然后点击**保存**。
报告及其参数将被保存以供将来使用。
