# 排除自己的访问

你想要访问自己的网站，但又不想让自己的访问在统计数据中显示出来。为了实现这一点，你需要在浏览器中添加一个设置。

在浏览器中打开开发者控制台：`设置 -> 更多工具 -> 开发者工具`

在控制台中输入以下代码，然后按下 `Enter` 键：

```javascript
localStorage.setItem('umami.disabled', 1);
```

这个设置是针对**每个网站**生效的，所以你需要对每个需要排除的网站都这样操作。

如果要移除这个设置，输入以下代码，然后按下 `Enter` 键：

```javascript
localStorage.removeItem('umami.disabled');
```
