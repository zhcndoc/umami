# 在 Netlify 上运行

[Netlify](https://netlify.com/) 提供了一个免费的托管服务，非常适合 GitHub 组织使用。

在这个设置中，你应该已经有一个正在运行并接受远程连接的数据库服务器。如果你还没有数据库，可以按照 [在 DigitalOcean 上运行](/docs/running-on-digitalocean) 指南来获取一个数据库并运行起来。你还可以在 [托管](/docs/guides/hosting) 下的 **托管数据库** 部分了解更多信息。

## 设置

[![使用 netlify 部署](https://www.netlify.com/img/deploy/button.svg)](https://app.netlify.com/start/deploy?repository=https://github.com/umami-software/umami)

_使用上面的按钮自动化执行步骤 1-5_

1. 在你的 GitHub 账户中 Fork [https://github.com/umami-software/umami](https://github.com/umami-software/umami) 项目。
2. 在 [Netlify](https://netlify.com/) 上创建一个账户。
3. 在仪表板页面上点击 **导入项目**，然后指定 GitHub 上你 fork 的项目的 URL。
4. 将构建命令更改为 **yarn run build**。
5. 添加所需的环境变量 `DATABASE_URL`。这些值在 [安装](/docs/install) 中的 **配置 Umami** 步骤中定义。
6. 部署并访问你的应用程序。
7. 从 [登录](/docs/login) 步骤开始遵循 **入门指南**，并确保更改默认密码。
