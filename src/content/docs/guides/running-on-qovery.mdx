# 在 Qovery 上运行

[Qovery](https://www.qovery.com) 是一个完全托管的云平台，运行在您的 AWS、Digital Ocean 和 Scaleway 帐户上，您可以在其中托管静态网站、后端 API、数据库、定时任务以及其他所有应用。

Qovery 为个人开发者提供**免费托管**，并包括以下功能：

- 从 GitHub 和 GitLab 进行持续自动构建和部署。
- 通过 [Let's Encrypt](https://letsencrypt.org) 进行自动 SSL 证书颁发。
- 免费托管的 PostgreSQL。
- 免费 SSD 存储。
- 无限的合作者数量。
- 无限的[自定义域名](https://hub.qovery.com/docs/using-qovery/configuration/application/#domains)。

## 设置

### 1. 创建 Qovery 账户

访问[Qovery 控制台](https://start.qovery.com)创建一个账户（如果您还没有）。

### 2. 创建项目

- 单击**创建项目**并为您的项目命名。
- 单击**下一步**。

![创建项目](https://hub.qovery.com/img/heroku/heroku-2.png)

### 3. 创建新环境

- 单击**创建环境**并给其命名（例如，staging, production）。

![创建新环境](https://hub.qovery.com/img/heroku/heroku-3.png)

### 4. 添加您的 Umami 应用

- 单击**创建应用程序**，给其命名，并选择您的 Umami 应用所在的 GitHub 或 GitLab 仓库。
- 定义主分支名称和根应用程序路径。
- 单击**创建**。

![添加应用程序](https://hub.qovery.com/img/rust/rust.png)

应用程序创建完成后：

- 转到您的应用程序**设置**。
- 选择**端口**。
- 添加您的 Umami 应用程序使用的端口。

### 5. 部署数据库

按照[此指南](https://hub.qovery.com/guides/getting-started/create-a-database)创建和部署一个新的 PostgreSQL 数据库。

### 6. 添加存储

要添加存储，转到您的应用程序**设置**：

![添加存储](https://hub.qovery.com/img/add-storage.png)

### 7. 设置您的 Umami 配置

要使用 Qovery 提供的 PostgreSQL，您可以使用内置的 secrets 和环境变量。您可以在我们的[配置部分](https://hub.qovery.com/docs/using-qovery/configuration/environment-variable/)中了解更多关于环境变量和 secrets 的信息。

### 8. 在 Qovery 上部署应用

现在，您只需转到您的应用程序，并单击**部署**。

![部署应用](https://hub.qovery.com/img/heroku/heroku-1.png)

就是这样。观察状态并等待应用程序部署完成。

要在浏览器中打开应用程序，请单击应用程序概览中的**操作**和**打开**。
