# 启用分享 URL

默认情况下，只有账户所有者可以查看网站的数据统计。但是你可以选择使用公共的 **分享 URL** 或通过 **团队** 来共享统计数据。

## 分享 URL

登录 Umami，点击页眉上的 **设置**。

<img src="/images/v2/navbar-settings.png" />

然后导航到 **网站** 并点击你的网站旁边的 **编辑** 按钮。

<img src="/images/v2/website-edit.png" />

导航到 **分享 URL** 选项卡，并切换为打开状态。

<img src="/images/v2/share-url.png" />

弹出的表单会显示你的 URL。

<img src="/images/v2/share-url-form.png" />

你可以点击 **重新生成** 按钮并保存来生成一个新的唯一的分享 URL 哈希值。
