# 添加用户

Umami 允许您创建单独的账户，以便您可以为朋友或客户提供数据托管服务。

以管理员身份登录 Umami，并点击顶部的**设置**。

<img src="/images/v2/navbar-settings.png" />

然后导航到**用户**并点击**创建用户**按钮。

<img src="/images/v2/user-create.png" />

填写表单详细信息并点击**保存**按钮。

<img src="/images/v2/user-create-form.png" />

新账户将拥有自己的仪表板和网站，与 **admin** 账户分开。

用户账户有两种可能的角色，**管理员**和**用户**。
只有**管理员**用户才能创建、修改和删除用户。普通**用户**账户将无法访问**用户**选项卡。
