# 添加网站

在 Umami 上登录并点击标题上的 **设置**。

<img src="/images/v2/navbar-settings.png" />

然后导航到 **网站** 并点击 **添加网站** 按钮。

<img src="/images/v2/website-add.png" />

填写表单详情并点击 **保存** 按钮。

<img src="/images/v2/website-add-form.png" />

**名称** 字段可以随意填写，通常与域名相同。

**域名** 字段是你网站的实际域名。它用于在指标中从转发者列表中筛选出你自己的网站。
