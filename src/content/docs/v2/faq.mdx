# 常见问题解答

### 1. Umami 是否符合 GDPR 要求？

是的，Umami 不会收集任何可识别个人身份的信息，并对所有收集的数据进行匿名化处理。用户不能被识别，也不会被跨网站跟踪。

### 2. 我是否需要向用户显示 cookie 通知？

不需要，Umami 在追踪代码中不使用任何 cookie。

### 3. Umami 是否适用于单页面应用 (SPA)？

是的，Umami 在 SPA 和普通网站之间可以无缝使用。

### 4. Umami 能否记录诸如按钮点击等事件？

是的，请在 **快速入门** 中查看 [跟踪事件](/docs/track-events)。

### 5. 如果我遇到问题或想要请求一个功能，我可以去哪里寻求帮助？

您可以在 [GitHub Discussions](https://github.com/umami-software/umami/discussions) 中进行一般讨论和故障排除。

对于错误报告和功能请求，请在 GitHub 上提交一个 [issue](https://github.com/umami-software/umami/issues)。

最有趣的是，加入 Umami 社区的 [Discord](https://discord.gg/4dz4zcXYrQ)！
