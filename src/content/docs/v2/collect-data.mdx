# 收集数据

要获取跟踪代码，请点击你想要跟踪的网站的**编辑**按钮。

<img src="/images/v2/website-edit.png" />

跟踪代码可以在**跟踪代码**选项卡下找到。

复制代码并将其插入到你网站的 `<head>` 部分。

<img src="/images/v2/website-tracking-code.png" />

然后访问你的网站，数据应该立即出现在 Umami 仪表板中。

### 故障排除

- 如果你的网站是使用 [Next.js](https://nextjs.org/) 构建的，请使用 [next/script](https://nextjs.org/docs/app/api-reference/components/script) 组件插入上述代码块。
