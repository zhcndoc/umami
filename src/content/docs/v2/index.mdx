# 什么是 Umami？

Umami 是一个简单、快速、注重隐私、开源的分析解决方案。Umami 是对 [Google Analytics](https://marketingplatform.google.com/about/analytics/) 的一个更好的替代品，因为它让你完全控制自己的数据，而且不会侵犯用户的隐私。

## 安装

请查看 [快速入门](/docs/quickstart) 部分，几分钟即可上手！

有几种不同的安装 Umami 的方法。

- **从源代码安装**：从 [GitHub](https://github.com/umami-software/umami) 获取代码并自行构建应用程序。参见 [安装](/docs/install)。
- **使用 Docker compose**：使用 `docker compose` 构建自己的 Docker 容器。参见 [安装](/docs/install)。
- **使用 Docker 镜像**：下载预构建的 Docker 镜像。参见 [安装](/docs/install)。

## 常见问题

**1. Umami 是否符合 GDPR？**

是的，Umami 不会收集任何可以识别个人身份的信息，并对所有收集的数据进行匿名化处理。用户无法被识别，并且不会在不同的网站之间进行追踪。

**2. 是否需要向用户显示 Cookie 通知？**

不需要，Umami 不会在跟踪代码中使用任何 cookie。

**3. Umami 是否适用于单页应用程序（SPA）？**

是的，Umami 在单页应用程序和普通网站之间无缝工作。

**4. Umami 能记录按钮点击等事件吗？**

可以，请参阅 **入门指南** 中的 [跟踪事件](/docs/track-events)。
