---
title: API
---

Umami 允许您通过调用应用程序的 API 端点直接提取数据。您在应用程序中可以执行的任何操作都可以在 API 中找到。所有数据以 JSON 格式返回。

### 自托管

在自托管时，API 端点可在 `http://<your-umami-instance>/api` 访问。在进行调用之前，您需要先进行 [身份验证](/docs/api/authentication)。

### Umami Cloud

如果使用 Umami Cloud，您可以使用 [API 密钥](/docs/cloud/api-key) 进行调用，调用地址为 `https://api.umami.is`。
