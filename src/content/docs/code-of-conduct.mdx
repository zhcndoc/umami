## 我们的承诺

作为社区的成员、贡献者和领导者，我们承诺让每个人在参与我们的社区时，都能获得无骚扰的体验，无论其年龄、体型、可见或不可见的残疾、种族、性特征、性别认同和表达、经验水平、教育程度、社会经济地位、国籍、个人外貌、种族、种姓、肤色、宗教或性身份和性取向如何。

我们承诺以促进开放、友好、多元、包容和健康的社区为目标来行动和互动。

## 我们的标准

促进我们社区积极环境的行为示例包括：

- 以同情和善良的态度对待他人
- 尊重不同的观点、看法和经历
- 给出并优雅地接受建设性的反馈
- 接受责任并向受到我们错误影响的人道歉，并从中学习
- 关注的不仅是我们个人的利益，还有整体社区的利益

不可接受行为的示例包括：

- 使用性别化语言或图像，以及任何形式的性关注或进攻
- 网络骚扰、侮辱性或贬损性评论，以及个人或政治攻击
- 公开或私下的骚扰
- 在未获得明确许可的情况下发布他人的私人信息，如地址或电子邮件地址
- 其他在专业环境中合理被认为不合适的行为

## 执行责任

社区领导者有责任明确和执行我们的可接受行为标准，并会对他们认定为不当、威胁、冒犯或有害的行为采取适当和公平的纠正行动。

社区领导者有权和责任删除、编辑或拒绝不符合本行为守则的评论、提交、代码、维基编辑、问题和其他贡献，并在适当情况下说明审查决定的理由。

### 范围

本行为守则适用于所有社区空间，并且在个人正式代表社区于公共空间时也适用。代表我们社区的示例包括使用官方电子邮件地址、通过官方社交媒体账户发布消息或作为在线或线下活动的指定代表。

### 执行

关于恶劣、骚扰或其他不可接受行为的事件，可以向负责执行的社区领导者举报，联系方式为 [contact@umami.is](mailto:contact@umami.is)。所有投诉将及时和公平地进行审核和调查。

所有社区领导者都有义务尊重报告任何事件的报告者的隐私和安全。

### 执行指南

社区领导者在决定对任何被认为违反本行为守则的行为的后果时，将遵循这些社区影响指南：

#### 1. 更正

社区影响：使用不当语言或其他被认为不专业或不受欢迎的行为。

后果：社区领导者下达的私人书面警告，提供关于违反性质的说明，以及为何该行为不当的解释。可能要求进行公开道歉。

#### 2. 警告

社区影响：通过单一事件或一系列行为的违反。

后果：对于持续行为的警告，警告后果为在特定时间段内不得与相关人员互动，包括未被请求的与执行行为守则之人互动。这包括避免在社区空间以及社交媒体等外部渠道中的互动。违反此条款可能会导致临时或永久禁令。

#### 3. 临时禁令

社区影响：严重违反社区标准，包括持续的不当行为。

后果：在特定时间段内，禁止与社区进行任何互动或公共沟通。在此期间，不允许与相关人员的任何公开或私下互动，包括未被请求与执行行为守则之人的互动。违反此条款可能会导致永久禁令。

#### 4. 永久禁令

社区影响：表现出违反社区标准的模式，包括持续的不当行为、对个人的骚扰，或对特定人群的攻击或贬损。

后果：禁止在社区内进行任何形式的公共互动。
