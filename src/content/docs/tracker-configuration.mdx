---
title: 追踪器配置
---

Umami 追踪器提供了多个属性，允许您配置其行为。

### data-host-url

默认情况下，Umami 将数据发送到脚本所在的位置。您可以覆盖此设置，将数据发送到其他位置。

用法：

```html
<script
  defer
  src="http://mywebsite.com/umami.js"
  data-website-id="94db1cb1-74f4-4a40-ad6c-962362670409"
  data-host-url="http://stats.mywebsite.com"
></script>
```

### data-auto-track

默认情况下，Umami 会自动跟踪所有页面浏览和事件。您可以禁用此行为，并使用 [追踪器函数](/docs/tracker-functions) 自行跟踪事件。

用法：

```html
<script
  defer
  src="http://mywebsite.com/umami.js"
  data-website-id="94db1cb1-74f4-4a40-ad6c-962362670409"
  data-auto-track="false"
></script>
```

### data-domains

如果您希望追踪器仅在特定域上运行，可以将这些域添加到您的追踪器脚本中。此列表以逗号分隔的域名形式展示。如果您在测试/开发环境中工作，这将非常有用。

用法：

```html
<script
  defer
  src="http://mywebsite.com/umami.js"
  data-website-id="94db1cb1-74f4-4a40-ad6c-962362670409"
  data-domains="mywebsite.com,mywebsite2.com"
></script>
```

### data-tag

如果您希望跟踪器在特定标签下收集事件。事件可以在仪表板中按特定标签过滤。

用法：

```html
<script
  defer
  src="http://mywebsite.com/umami.js"
  data-website-id="94db1cb1-74f4-4a40-ad6c-962362670409"
  data-tag="umami-eu"
></script>
```

### data-exclude-search

如果您不想从 URL 中收集搜索参数。

用法：

```html
<script
  defer
  src="http://mywebsite.com/umami.js"
  data-website-id="94db1cb1-74f4-4a40-ad6c-962362670409"
  data-exclude-search="true"
></script>
```

### data-exclude-hash

如果您不想从 URL 中收集哈希值。

用法：

```html
<script
  defer
  src="http://mywebsite.com/umami.js"
  data-website-id="94db1cb1-74f4-4a40-ad6c-962362670409"
  data-exclude-hash="true"
></script>
```

### data-do-not-track

尊重用户的 [不追踪](https://developer.mozilla.org/en-US/docs/Web/API/Navigator/doNotTrack) 浏览器设置。

用法：

```html
<script
  defer
  src="http://mywebsite.com/umami.js"
  data-website-id="94db1cb1-74f4-4a40-ad6c-962362670409"
  data-do-not-track="true"
></script>
```
