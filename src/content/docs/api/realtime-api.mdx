您网站的实时数据。

**端点**

```text
GET /api/realtime/:websiteId
```

---

## GET /api/realtime/:websiteId

获取网站实时数据。

**参数**

- `startAt`：（数字）起始日期的时间戳（毫秒级）。
- `endAt`：（数字）结束日期的时间戳（毫秒级）。
- `timezone`：（字符串）时区。
