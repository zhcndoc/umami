---
title: 网站
---

与网站管理和统计相关的操作。

**端点**

```text
GET /api/websites
POST /api/websites
GET /api/websites/:websiteId
POST /api/websites/:websiteId
DELETE /api/websites/:websiteId
POST /api/websites/:websiteId/reset
```

---

## GET /api/websites

返回所有被跟踪的网站。

**参数**

- `query`: （可选字符串）搜索文本。
- `page`: （可选数字，默认值 1）确定页码。
- `pageSize`: （可选字符串）确定返回多少结果。
- `orderBy`: （可选字符串，默认值 `name`）按列名排序。

**示例响应**

```json
{
    "data": [
        {
            "id": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
            "name": "Example",
            "domain": "example.com",
            "shareId": null,
            "resetAt": null,
            "userId": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
            "teamId": null,
            "createdBy": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
            "createdAt": "0000-00-00T00:00:00.000Z",
            "updatedAt": "0000-00-00T00:00:00.000Z",
            "deletedAt": null,
            "user": {
                "username": "xxxxxxx@xxxx.xxxx",
                "id": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx"
            }
        }
    ],
    "count": 2,
    "page": 1,
    "pageSize": 10,
    "orderBy": "name"
}


```

---

## POST /api/websites

创建一个网站。

**参数**

- `domain`: （字符串）被跟踪网站的完整域名。
- `name`: （字符串）Umami 中的网站名称。
- `shareId`: （字符串|可选）一个唯一字符串以启用共享 URL。设置为 `null` 以取消共享。
- `teamId`: （字符串|可选）创建网站所在团队的 ID。


**请求体**

```json 
{
    "name": "Test",
    "domain": "example.com"
}
```

**示例响应**

```json
{
    "id": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
    "name": "Test",
    "domain": "example.com",
    "shareId": null,
    "resetAt": null,
    "userId": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
    "teamId": null,
    "createdBy": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
    "createdAt": "0000-00-00T00:00:00.000Z",
    "updatedAt": "0000-00-00T00:00:00.000Z",
    "deletedAt": null
}
```

## POST /api/websites/:websiteId

更新网站。

**参数**

- `name`: (string) Umami 网站的名称。
- `domain`: (string) 被追踪网站的完整域名。
- `shareId`: (string | optional) 用于启用共享 URL 的唯一字符串。设置为 `null` 以取消共享。

**请求主体**

```json
{
    "name": "Test",
    "domain": "domain.com"
}
```

**示例响应**

```json
{
    "id": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
    "name": "Test",
    "domain": "domain.com",
    "shareId": null,
    "resetAt": null,
    "userId": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
    "teamId": null,
    "createdBy": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
    "createdAt": "0000-00-00T00:00:00.000Z",
    "updatedAt": "0000-00-00T00:00:00.000Z",
    "deletedAt": null
}
```
