---
title: 网站
---

Operations around Website management and statistics.

**端点**

```text
GET /api/websites
POST /api/websites
GET /api/websites/:websiteId
POST /api/websites/:websiteId
DELETE /api/websites/:websiteId
POST /api/websites/:websiteId/reset
```

---

## GET /api/websites

返回所有的跟踪网站。

**参数**

- `query`: （可选字符串）搜索文本。
- `page`: （可选数字，默认值为1）确定页码。
- `pageSize`: （可选字符串）确定要返回的结果数量。
- `orderBy`: （可选字符串，默认为`name`）按列名排序。

**示例响应**

```json
[
  {
    "id": "02d89813-7a72-41e1-87f0-8d668f85008b",
    "name": "我的网站",
    "domain": "mywebsite.com",
    "shareId": null,
    "resetAt": null,
    "websiteId": "1a457e1a-121a-11ee-be56-0242ac120002",
    "createdAt": "2023-04-10T23:06:44.250Z",
    "updatedAt": null,
    "deletedAt": null
  }
]
```

---

## POST /api/websites

创建网站。

**参数**

- `domain`: （字符串）跟踪网站的完整域名。
- `name`: （字符串）在 Umami 中的网站名称。
- `shareId`: （可选字符串）用于启用共享 URL 的唯一字符串。设置为 `null` 取消共享。
- `teamId`: （可选字符串）将创建网站的团队的 ID。

**示例响应**

```json
{
  "id": 4,
  "websiteUuid": "51f73213-3f01-4343-a135-25496a3ffd31",
  "websiteId": 2,
  "name": "Umami",
  "domain": "umami.is",
  "shareId": "8PWex1pa",
  "createdAt": "2021-07-26T17:17:52.846Z"
}
```

```json
  "createdAt": "2023-04-10T23:06:44.250Z",

```
