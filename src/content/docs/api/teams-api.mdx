# 团队

团队管理操作。

**端点**

```text
POST /api/teams
GET /api/teams
POST /api/teams/join
POST /api/teams/:teamId
GET /api/teams/:teamId
DELETE /api/teams/:teamId
POST /api/teams/:teamId/users
GET /api/teams/:teamId/users
GET /api/teams/:teamId/users/:userId
POST /api/teams/:teamId/users/:userId
DELETE /api/teams/:teamId/users/:userId
GET /api/teams/:teamId/websites
```

---

## POST /api/teams

创建一个团队。

**参数**

- `name`: (string) 团队的名称。

**示例响应**

```json
[
  {
    "accessCode": "xxwtoY8pzKjDIUQi",
    "createdAt": "2023-04-13T20:22:55.756Z",
    "id": "ae09865c-ff82-4c43-acba-dd1e6540cda5",
    "name": "My Team",
    "updatedAt": null
  },
  {
    "createdAt": "2023-04-13T20:22:55.756Z",
    "id": "a9b1fbbc-ac22-4422-aa74-b2a2751ad87d",
    "role": "team-owner",
    "teamId": "ae09865c-ff82-4c43-acba-dd1e6540cda5",
    "updatedAt": null,
    "userId": "1a457e1a-121a-11ee-be56-0242ac120002"
  }
]
```

## GET /api/teams

返回所有团队。

**参数**

- `query`: (可选 string) 搜索文本。
- `page`: (可选 number, 默认为 1) 确定页码。
- `pageSize`: (可选 string) 确定返回结果的数量。
- `orderBy`: (可选 string, 默认为 `name`) 按列名排序。

**示例响应**

```json
[
  {
    "accessCode": "xxwtoY8pzKjDIUQi",
    "createdAt": "2023-04-13T20:22:55.756Z",
    "id": "ae09865c-ff82-4c43-acba-dd1e6540cda5",
    "name": "My Team",
    "teamUser": [
      {
        "createdAt": "2023-04-13T20:22:55.756Z",
        "id": "a9b1fbbc-ac22-4422-aa74-b2a2751ad87d",
        "role": "team-owner",
        "teamId": "ae09865c-ff82-4c43-acba-dd1e6540cda5",
        "updatedAt": null,
        "user": { "id": "1a457e1a-121a-11ee-be56-0242ac120002", "username": "admin" },
        "userId": "1a457e1a-121a-11ee-be56-0242ac120002"
      }
    ]
  }
]
```

## GET /api/teams/:teamId/users

获取属于一个团队的所有用户。

**参数**

- `query`: (可选 string) 搜索文本。
- `page`: (可选 number, 默认为 1) 确定页码。
- `pageSize`: (可选 string) 确定返回结果的数量。
- `orderBy`: (可选 string, 默认为 `name`) 按列名排序。

**示例响应**

```json
[
  {
    "id": "a9b1fbbc-ac22-4422-aa74-b2a2751ad87d",
    "teamId": "ae09865c-ff82-4c43-acba-dd1e6540cda5",
    "userId": "1a457e1a-121a-11ee-be56-0242ac120002",
    "role": "team-owner",
    "createdAt": "2023-04-13T20:22:55.756Z",
    "updatedAt": null,
    "user": {
      "id": "1a457e1a-121a-11ee-be56-0242ac120002",
      "username": "admin"
    }
  }
]
```

---

## POST /api/teams/:teamId/users

将用户添加到一个团队。

**参数**

- `userId`: 要添加的用户的 ID。
- `role`: 用户将被添加为的角色 (`member` 或 `view-only`)。

**示例响应**

```json
{
  "id": "a9b1fbbc-ac22-4422-aa74-b2a2751ad87d",
  "teamId": "ae09865c-ff82-4c43-acba-dd1e6540cda5",
  "userId": "1a457e1a-121a-11ee-be56-0242ac120002",
  "role": "team-owner",
  "createdAt": "2023-04-13T20:22:55.756Z",
  "updatedAt": null
}
```
