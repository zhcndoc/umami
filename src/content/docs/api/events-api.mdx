关于事件及事件数据的操作。

**端点**

```text
GET /api/websites/:websiteId/events
GET /api/websites/:websiteId/event-data/events
GET /api/websites/:websiteId/event-data/fields
GET /api/websites/:websiteId/event-data/values
GET /api/websites/:websiteId/event-data/stats
```

---

## GET /api/websites/:websiteId/events

获取给定时间范围内的网站事件详细信息。

**参数**

- `startAt`: 开始日期的时间戳（以毫秒为单位）。
- `endAt`: 结束日期的时间戳（以毫秒为单位）。
- `query`: （字符串 | 字符串）搜索文本。
- `page`: （数字 | 可选，默认为 1）确定页码。
- `pageSize`: （字符串 | 可选）确定要返回多少结果。
- `orderBy`: （字符串 | 可选）按列名排序。

## GET /api/websites/:websiteId/event-data/events

获取事件数据名称、属性和计数。

**参数**

- `startAt`: 开始日期的时间戳（毫秒）。
- `endAt`: 结束日期的时间戳（毫秒）。
- `event`: （可选）事件名称过滤。

**示例响应**

```json
[
  {
    "eventName": "button-click",
    "propertyName": "id",
    "dataType": 1,
    "total": 4
  },
  {
    "eventName": "button-click",
    "propertyName": "name",
    "dataType": 1,
    "total": 4
  },
  {
    "eventName": "track-product",
    "propertyName": "price",
    "dataType": 2,
    "total": 2
  }
]
```

## GET /api/websites/:websiteId/event-data/fields

获取给定时间范围内事件数据属性和值的计数。

**参数**

- `websiteId`: （uuid）网站的唯一标识符。
- `startAt`: 开始日期的时间戳（毫秒）。
- `endAt`: 结束日期的时间戳（毫秒）。

**示例响应**

```json
[
  {
    "propertyName": "age",
    "dataType": 2,
    "value": "33",
    "total": 1
  },
  {
    "propertyName": "age",
    "dataType": 2,
    "value": "31",
    "total": 4
  },
  {
    "propertyName": "gender",
    "dataType": 1,
    "value": "female",
    "total": 4
  },
  {
    "propertyName": "gender",
    "dataType": 1,
    "value": "male",
    "total": 1
  }
]
```

---

## GET /api/websites/:websiteId/event-data/values

获取给定事件和属性的事件数据计数。

**参数**

- `startAt`: 开始日期的时间戳（毫秒）。
- `endAt`: 结束日期的时间戳（毫秒）。
- `eventName`: 事件名称。
- `propertyName`: 属性名称。

**示例响应**

```json
[
  { "value": "female", "total": 4 },
  { "value": "male", "total": 1 }
]
```

---

## GET /api/websites/:websiteId/event-data/stats

获取给定时间范围内总结的网站事件、字段和记录。

**参数**

- `startAt`: 开始日期的时间戳（毫秒）。
- `endAt`: 结束日期的时间戳（毫秒）。

**示例响应**

```json
[
  {
    "events": 16,
    "fields": 13,
    "records": 26
  }
]
```
