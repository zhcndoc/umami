关于事件及事件数据的操作。

**端点**

```text
GET /api/websites/:websiteId/events
GET /api/websites/:websiteId/event-data/:eventId
GET /api/websites/:websiteId/event-data/events
GET /api/websites/:websiteId/event-data/fields
GET /api/websites/:websiteId/event-data/properties
GET /api/websites/:websiteId/event-data/values
GET /api/websites/:websiteId/event-data/stats
```

---

## Filters

所有带有 `filters` 标记的端点现在可以使用以下参数进行过滤。

**参数**

- `path`: (string | optional) URL 名称。
- `referrer`: (string | optional) 引荐来源名称。
- `title`: (string | optional) 页面标题名称。
- `query`: (string | optional) 查询参数名称。
- `browser`: (string | optional) 浏览器名称。
- `os`: (string | optional) 操作系统名称。
- `device`: (string | optional) 设备名称（例如 Mobile）
- `country`: (string | optional) 国家名称。
- `region`: (string | optional) 地区/州/省名称。
- `city`: (string | optional) 城市名称。
- `hostname`: (string | optional) 主机名。
- `tag`: (string | optional) 标签名称。
- `segment`: (uuid | optional) 片段 UUID。
- `cohort`: (uuid | optional) 群体 UUID。

## GET /api/websites/:websiteId/events

**参数**

- `startAt`: (number) 开始日期的时间戳（毫秒）。
- `endAt`: (number) 结束日期的时间戳（毫秒）。
- `search`: (string | optional) 搜索文本。
- `page`: (optional number, 默认为 1) 页码。

## GET /api/websites/:websiteId/event-data/events

获取事件数据名称、属性和计数。

**参数**

- `startAt`: (number) 开始日期的时间戳（毫秒）。
- `endAt`: (number) 结束日期的时间戳（毫秒）。
- `event`: (string | optional) 事件名称过滤。
- `filters`: 可接受过滤参数。

**示例响应**

```json
[
  {
    "eventName": "button-click",
    "propertyName": "id",
    "dataType": 1,
    "total": 4
  },
  {
    "eventName": "button-click",
    "propertyName": "name",
    "dataType": 1,
    "total": 4
  },
  {
    "eventName": "track-product",
    "propertyName": "price",
    "dataType": 2,
    "total": 2
  }
]
```

    "dataType": 2,
    "value": "33",

    "value": "31",
    "total": 4
  },
  {
    "propertyName": "gender",
    "dataType": 1,
    "value": "female",
    "total": 4
  },
  {
    "propertyName": "gender",
    "dataType": 1,
    "value": "male",
    "total": 1
  }
]
```

  },

    "eventName": "revenue-demo",
    "propertyName": "currency",
    "total": 122
  }
]
```

## GET /api/websites/:websiteId/event-data/values

获取给定事件和属性的事件数据计数。

**参数**

- `startAt`: (number) 开始日期的时间戳（毫秒）。
- `endAt`: (number) 结束日期的时间戳（毫秒）。
- `event`: (string) 事件名称。
- `propertyName`: (string) 属性名称。
- `filters`: 可接受过滤参数。

**示例响应**

```json
[
  {
    "value": "Male",
    "total": 28
  },
  {
    "value": "Female",
    "total": 26
  }
]
```

## GET /api/websites/:websiteId/event-data/stats

获取给定时间范围内汇总的网站事件、属性和记录。

**参数**

- `startAt`: (number) 开始日期的时间戳（毫秒）。
- `endAt`: (number) 结束日期的时间戳（毫秒）。
- `filters`: 可接受过滤参数。

**示例响应**

```json
[
  {
    "events": 16,
    "properties": 13,
    "records": 26
  }
]
```
