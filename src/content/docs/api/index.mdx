# API

Umami 允许您通过调用应用程序的 API 端点直接获取数据。

如果自行托管，则 API 端点可在 `http://<your-umami-instance>/api` 上访问。在发出调用之前，您首先需要进行[身份验证](/docs/api/authentication)。

如果使用 Umami Cloud，则可以使用 [API 密钥](/docs/cloud/api-key) 在 `https://api.umami.is` 上进行调用。

您可以通过应用程序执行的任何操作都可以在 API 中进行。所有数据都以 JSON 形式返回。
