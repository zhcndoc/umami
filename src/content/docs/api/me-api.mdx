---
title: 我
---

关于您会话的所有信息。

**端点**

```text
GET /api/me
GET /api/me/teams
GET /api/me/websites
```

---

## GET /api/me

根据您的身份验证令牌获取信息。

**示例响应**

```json
{
    "user": {
        "id": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
        "username": "xxxxxxxxxx@xxxxxx.xxxx",
        "role": "xxxx",
        "createdAt": "0000-00-00T00:00:00.000Z",
        "isAdmin": false
    },
    "grant": [],
    "token": "xxxxxxxxxxxxxxx",
    "shareToken": null
}
```
