# 发送统计数据

## POST /api/send

要注册一个“事件”，你需要向 `/api/send` 发送一个 `POST` 请求，请求数据如下：

**参数**

`payload`

- `hostname`: (字符串) 主机名。
- `language`: (字符串) 访客的语言 (例如 "en-US")。
- `referrer`: (字符串) 引荐网址。
- `screen`: (字符串) 屏幕分辨率 (例如 "1920x1080")。
- `title`: (字符串) 页面标题。
- `url`: (字符串) 页面网址。
- `website`: (字符串) 网站 ID。
- `name`: (字符串) 事件名称。
- `data`: (对象)(可选) 事件的附加数据。

`type`: (字符串) 目前只有 `event` 类型可用。

**示例请求数据**

```json
{
  "payload": {
    "hostname": "your-hostname",
    "language": "en-US",
    "referrer": "",
    "screen": "1920x1080",
    "title": "dashboard",
    "url": "/",
    "website": "your-website-id",
    "name": "event-name",
    "data": {
      "foo": "bar"
    }
  },
  "type": "event"
}
```

注意，在 `/api/send` 请求中，你不需要发送身份验证令牌。

另外，你需要发送一个正确的 `User-Agent` HTTP 头部，否则你的请求将无法被注册。

**编程方式**

你可以使用浏览器 API 通过 JavaScript 编程生成大部分这些值。例如：

```js
const data = {
  payload: {
    hostname: window.location.hostname,
    language: navigator.language,
    referrer: document.referrer,
    screen: `${window.screen.width}x${window.screen.height}`,
    title: document.title,
    url: window.location.pathname,
    website: 'your-website-id',
    name: 'event-name',
  },
  type: 'event',
};
```
