---
title: 收入
---

Umami 收入功能帮助您跟踪财务表现、衡量投资回报率 (ROI)，并识别哪些页面或产品产生了最多收入。它提供了对客户行为的洞察，帮助优化收入来源并提高转化率。通过将收入数据与流量和用户行为关联，企业可以做出明智的决策，提高盈利能力并规划未来增长。

该洞察通过汇总指定时间段内的**收入**和**货币**数据实现。

## 配置收入

要开始收集收入数据，您需要跟踪带有动态数据的事件。这可以通过[追踪器函数](https://umami.is/docs/tracker-functions#event-data)或[数据属性](https://umami.is/docs/track-events#using-data-attributes)实现。被跟踪的事件必须传递两个动态数据属性 `revenue` 和 `currency`。如果货币代码（ISO 4217）无法识别，该洞察将默认为 `USD`。

### 追踪器函数

```js
umami.track('checkout-cart', { revenue: 19.99, currency: 'USD' });
```

跟踪事件时，默认属性包含在负载中。
这等同于执行：

```js
umami.track(props => ({
  ...props,
  name: 'checkout-cart',
  data: {
    revenue: 19.99,
    currency: 'USD',
  },
}));
```

### 数据属性

请注意，使用此方法时，收入必须作为字符串传递。

```html
<button
  id="signup-button"
  data-umami-event="checkout-cart"
  data-umami-event-revenue="19.99"
  data-umami-event-currency="USD"
>
  结账
</button>
```

## 参数

- `Currency`：（必填）要汇总的数据的货币。
