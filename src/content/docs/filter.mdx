---
title: 过滤器
---

Umami 内置了报告功能，帮助您深入了解数据的各个部分。通过通用过滤，您可以在所有页面/报告中应用条件，实现一致的分析。  
分段功能允许您根据共同的属性或行为对用户进行分组。群组分析进一步通过随时间分析用户组，帮助您跟踪留存率、表现和长期趋势。

## 可用过滤器

- **路径** - 访问页面的 URL 路径。
- **页面标题** - 浏览器标签中显示的 HTML 页面标题。
- **推荐来源** - 引导访客访问您网站的来源。
- **浏览器** - 访客使用的 web 浏览器（如 Chrome、Safari、Firefox）。
- **操作系统** - 访客设备的操作系统（如 Windows、macOS、iOS、Android）。
- **设备** - 使用的设备类型，如台式机、平板或手机。
- **国家** - 基于地理位置的访客国家。
- **地区** - 访客所在的州、省或地区。
- **城市** - 访客所在城市。
- **主机名** - 访问页面所在的域名。
- **标签** - 您为数据分配的标签或关键词。
- **事件** - 用户触发的自定义事件（如按钮点击、表单提交、视频播放）。

## 通用过滤

网页页面中的日期范围和过滤器现在可以应用于所有页面和报告。

## 访问过滤器

登录 Umami 后，点击侧边栏中的 **网站**。

![image](/images/docs/v3/navbar.png)

点击您想查看的网站的 **查看** 按钮。

![image](/images/docs/v3/website-edit.png)

屏幕顶部将显示可以应用于所有页面的日期和字段过滤器。

![image](/images/docs/v3/filter-access.png)

## 添加过滤器

**方法一：** 点击 **过滤器** 按钮来应用过滤条件。

![image](/images/docs/v3/filter-add.png)

**方法二：** 在概览页面的指标卡上点击任意行。

![image](/images/docs/v3/filter-card.png)

当前应用于每个页面的过滤条件会显示在页面顶部。

![image](/images/docs/v3/filter-details.png)
