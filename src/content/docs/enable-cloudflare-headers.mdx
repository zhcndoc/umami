---
title: 启用 Cloudflare 头部信息
---

如果您在网站中使用了 Cloudflare，Umami 会使用 Cloudflare 发送的头部信息来确定访问者的位置信息。然而，默认情况下 Cloudflare 仅发送国家的数据。如果您还需要区域和城市信息，则需要配置 Cloudflare 以发送额外的头部信息。

## 步骤

1. 登录 Cloudflare 控制面板，选择您的账户和网站。

2. 进入 规则 > 设置。

3. 进入 管理的转换（Managed Transforms）选项卡。

4. 启用 **添加访问者位置信息头部（Add visitor location headers）** 设置。
